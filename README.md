# DMC Ventures PH

DMC Ventures PH is a corporate landing website concept for an umbrella business brand focused on property, commerce, and technology.

## Business Divisions

### DMC Realty
Real estate brokerage, property marketing, and client advisory.

### DMC Trading
Consumer goods sourcing, wholesale, and merchandise trading.

### DMC Labs
Websites, apps, automation, CRM tools, and digital business systems.

## Project Overview

This website is built as a simple static frontend using:

- HTML
- CSS
- JavaScript

The goal of this first version is to create a clean, corporate online presence that introduces the DMC Ventures PH brand and its three business divisions.

## Pages

- `index.html` — Main DMC Ventures PH homepage
- `realty.html` — DMC Realty landing page
- `trading.html` — DMC Trading landing page
- `labs.html` — DMC Labs landing page
